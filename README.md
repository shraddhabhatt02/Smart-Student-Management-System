# Smart-Student-Management-System
A Python-based Student Management System built using Object-Oriented Programming.

## Features

- Add Student
- View Student
- Search Student
- Update Marks
- Delete Student
- Add Teacher
- View Teacher
- Calculate Average Marks
- Find Highest Scorer
- Find Lowest Scorer
- Pass/Fail Analysis
- Save and Load Student Data

## Concepts Used

- OOP (Inheritance, Method Overriding)
- Exception Handling
- File Handling
- Functions
- Lists

## Technologies

- Python 3
